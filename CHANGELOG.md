# Changelog

## 1.0.0 - 2026-07-07

- Packaged the Windows single-file desktop build.
- Added local WinForms interface for drag-and-drop video processing.
- Added full-length handling by segmenting videos that exceed the preview
  engine's time or frame-count limits.
- Added validation for output resolution, frame rate, duration, frame count,
  and audio preservation.
- Added provenance sidecar generation.
- Added third-party component notes and release checksums.
