# Notice

This repository contains wrapper scripts, a Windows desktop launcher, packaging
code, and documentation for a local visible-mark repair workflow.

The source repository intentionally does not include runtime binaries. The v1.0
release package bundles third-party runtime components documented in
`THIRD_PARTY.md`.

This project is intended only for videos the user is authorized to edit. It
does not detect, remove, or modify Google SynthID or any other invisible
provenance watermark.
