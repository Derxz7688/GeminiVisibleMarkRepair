功能亮点 / Highlights

为不习惯命令行的使用方式做了桌面封装，底层仍保留可审计的脚本和来源说明。

单文件启动

解压后双击 EXE 使用，首次运行自动释放内置运行时。

本机处理

视频文件只在本机处理，不上传到外部服务。

自动分段

超过上游预览引擎帧数限制时自动分段、修复、拼接。

音轨保留

输出尽量保持原始分辨率、帧率、时长、帧数和音轨。

来源记录

输出 `.provenance.json`，记录处理说明与参数。

边界清楚

仅处理可见角标，不提供 SynthID 绕过能力。

使用边界 / Usage boundary: 请只处理你有权编辑的视频。v1.0 已实测横屏 Gemini 可见角标；Veo、竖屏、非标准位置和复杂遮挡场景尚未完成实测，请视为实验性兼容范围。
